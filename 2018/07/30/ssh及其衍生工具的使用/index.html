ssh及其衍生工具的使用

发表于 2018-07-30 | 分类于 Tools

SSH 简介

SSH 为 Secure Shell 的缩写，由 IETF 的网络小组（Network Working Group）所制定；SSH为建立在应用层基础上的安全协议。SSH是目前较可靠，专为远程登录会话和其他网络服务提供安全性的协议。利用 SSH 协议可以有效防止远程管理过程中的信息泄露问题。SSH最初是UNIX系统上的一个程序，后来又迅速扩展到其他操作平台。SSH在正确使用时可弥补网络中的漏洞。SSH客户端适用于多种平台。几乎所有UNIX平台—包括HP-UX、Linux、AIX、Solaris、Digital UNIX、Irix，以及其他平台，都可运行SSH。

摘自百度百科ssh词条

SSH 使用

SSH 是通过网络连接其他的PC，实现远程登录和其他的远程操作，所以使用SSH意味着需要至少两台设备，一台设备上运行ssh server服务，一台设备运行ssh client程序。目前SSH协议实现软件openssh使用的较为广泛，下面的内容也是基于openssh，分别介绍openssh-server和openssh-client（平台使用的是Ubuntu）

server

server指的是SSH的连接对象。上面必须运行有openssh-server

  1. ssh server 的安装
    Ubuntu系统上一般使用openssh，openssh是SSH协议的免费开源的实现。直接使用apt命令就可以安装。

  2. 运行openssh-server
    安装好openssh-server之后软件已经自动运行起来了，通过service命令来查看它的运行情况以及开启，关闭它。

  3. 查看openssh-server的运行状态,注意service命令后面的服务名是ssh，这是因为在openssh-server运行时，创建的服务名称叫ssh。

从第4行可以看出ssh服务的运行状态是running，表示正在运行，如果是dead表示停止。10行和11行显示的是ssh服务占用的端口和ip，::表示的是监听了本机拥有的ip。通过ifconfig可以查看本机的ip地址。

  1. 启动，重启，关闭ssh服务

  2. 修改ssh服务运行的端口
    默认的ssh服务占用的端口是22，如果是个人或者非公开使用，为了安全起见，最好修改端口号。
